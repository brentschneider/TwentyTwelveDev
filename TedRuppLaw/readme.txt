== Description ==
A flexible theme by ITstar with sticky widgets and jquery effects.

Requires WordPress 3.6.1 and PHP 5.

**Current Features**
* Responsive (200px to 1600px Screen view)
* Manual Header Image
* Manual Background Image & Color
* Widget Area on sidebar 
* 3 Widget Area on Footer
* A Navigation Widget
* Support Goolge Fonts
* Header Menu
* Responsive Menu
* 8 Menu Colors
* 5 Responsive Menu Effects
* HTML5 and CSS3 Validated
* SEO Optimized

== Installation ==

1. Upload 'persia' to the '/wp-content/themes/' directory,
2. Activate the theme through the 'theme' menu in WordPress.
3.Install Header Menu and Responsive Menu.
